# Lab5 – Text Analyzer App

## 📌 Опис
Програма на C# (Windows Forms), яка підраховує кількість входжень кожного символу в текстовому файлі.

Користувач обирає файл, після чого програма:
- аналізує всі символи
- підраховує їх кількість
- записує результат у файл `result.txt`

---

## ⚙️ Функціонал
- Вибір текстового файлу через форму
- Підрахунок кожного символу (включаючи пробіли)
- Вивід статистики у файл
- Обчислення загальної кількості букв

---
## ▶️ Як запустити

```bash
cd Lab5/TextAnalyzerApp
dotnet run --project TextAnalyzerApp.csproj
```

## 🧪 Приклад результату
a - 5
b - 2
c - 7
(space) - 11
Загальна кількість букв: 49
