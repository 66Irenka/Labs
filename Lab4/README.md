# Lab4 – File Handling App

## 📌 Опис
Програма на C# (Windows Forms), яка працює з файлами: створює, зчитує та записує дані.
У лабораторній роботі реалізовано роботу з файловою системою та окремою папкою для даних.

---

## ⚙️ Функціонал
- Створення файлів програмно
- Перевірка існування файлів
- Зчитування тексту з файлів
- Запис результатів у файли
- Робота з директоріями

---

## ▶️ Як запустити

```bash
dotnet run --project Lab4/QuestionsApp/QuestionsApp.csproj
```

## 📂 Дані
Файли для роботи програми знаходяться в папці:
Lab4/Lab4Data/
Програма читає та записує дані саме в цю папку.

## 🧪 Логіка роботи
Перевіряється наявність файлів
Якщо файл відсутній — він створюється
Зчитується вміст файлу
Виконується обробка
Результат записується у новий файл
