## 🚀 Про репозиторій

Цей репозиторій — це структуроване портфоліо лабораторних робіт, виконаних під час навчання. Основний фокус зосереджений на створенні Desktop-додатків, обробці даних та алгоритмах захисту інформації.

### Що реалізовано:
- 🔐 **Криптографія:** Алгоритми шифрування (Цезар, робота з паролями).
- 📂 **Файлова система:** Глибока робота з I/O та потоками даних.
- 🖼 **UI/UX:** Інтерактивні інтерфейси на Windows Forms.
- 📊 **Аналітика:** Обробка та статистичний аналіз текстових масивів.

## 🧪 Labs Overview

| Лабораторна | Назва проекту | Ключові функції |
| :--- | :--- | :--- |
| **Lab 2** | `Caesar Cipher` | Шифрування та дешифрування тексту методом зсуву. |
| **Lab 3** | `Secure Crypt` | Захист даних за допомогою пароля та складніших ключів. |
| **Lab 4** | `File Explorer` | Windows Forms додаток для маніпуляцій з файлами. |
| **Lab 5** | `Text Analyzer` | Глибокий аналіз частоти символів у тексті. |
